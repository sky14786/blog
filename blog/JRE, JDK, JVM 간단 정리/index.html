JRE, JDK, JVM 간단 정리

February 05, 2022
JavaJDKJREJVM

  • JRE(Java Runtime Environment)

    • Java 언어로 작성된 프로그램을 실항하기 위한 환경을 제공한다.
    • JAVA API와 JVM으로 구성되어있다.

  • JDK(Java Development Kit)

    • Java 언어로 프로그래밍을 할수 있게 도와주는 도구이다.

  • JVM(Java Virtual Machine)

    • 물리적인 환경을 소프트웨어로 구현한 것이다.
    • OS와 Java로 작성된 프로그램의 중간에 위치한다.
    • Java언어로 작성된 프로그램이 OS에 구애받지 않고 사용을 가능하게 한다.
    • 스택기반의 가상머신이다.
    • OS로부터 할당받은 메모리 관리 및 GC(Garbage Collection)을 수행한다.

  • 자바 프로그램 실행 과정

    1. JVM이 OS로부터 필요한 메모리 용량을 할당받는다.
    2. 할당받은 메모리를 용도에 따라 분리한다.
    3. 자바 컴파일러(javac)가 자바 소스코드를(Java)를 읽어들여 바이트 코드로(.class)로 변환한다.
    4. Class Loader를 통해 Class파일들을 JVM으로 로딩한다.
    5. 로딩된 Class파일들은 Execution engine을 통해 해석된다.
    6. 해석된 바이트 코드는 Runtime Data Areas에 배치되어 실질적인 작동이 이루어진다.
