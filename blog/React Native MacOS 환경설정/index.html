React-Native (MacOS) 환경설정

October 19, 2021

사전 준비 도구

  • Homebrew
  • Watchman
  • Node Version Manager(NVM)
  • Node Package Manager(NPM)
  • Xcode
  • Android Studio

1. Homebrew Install

  1. 터미널을 열어 아래의 명령어를 입력하여 homebrew를 설치한다.

    • $ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  2. 설치가 끝난 후 아래와 같이 Next step이 나타나면 로그의 명령어를 그대로 터미널에 입력한다.

    • 설치 후 로그
==> Next steps:
- Add Homebrew to your PATH in /Users/<USER_ID>/.zprofile:
    echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> /Users/<USER_ID>/.zprofile
    eval "$(/opt/homebrew/bin/brew shellenv)"
- Run `brew help` to get started
- Further documentation:
    https://docs.brew.sh
  • 입력할 명령어
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> /Users/<USER_ID>/.zprofile
eval "$(/opt/homebrew/bin/brew shellenv)"

2. Watchman Install

  • Watchman은 페이스북에서 제공하는 툴로 파일시스템의 변경사항을 감지한다.****
  • React native에서는 소스코드 변경이 발생하면 Watchman이 감지하여 재로딩하여 다시 빌드하지 않고 변경사항을 확인 할 수 있다.
  • 터미널에 아래의 명령어를 입력한다.
  • brew insatall watchman

3. Node Version Manger(NVM), Node Package Manger(NPM), React-native CLI 설치

  • NVM을 설치하기 앞서 Homebrew라는 도구가 필요한데 해당 도구의 설치방법은 위의 명령어를 참고하면 되며, 자세한 사항은 링크를 참조하기 바란다.
  • 터미널에 brew install nvm 을 입력한다.
  • 터미널에 vi ~/.zshrc 또는 open ~/.zshrc 를 이용하여 편집기를 열어준다.

  • 아래의 환경변수를 입력한 후 저장한다.

    # NVM
export NVM_DIR=~/.nvm
source $(brew --prefix nvm)/nvm.sh
  • source ~/.zshrc 를 입력하여 환경변수를 적용해 준다.
  • nvm install {Node Version} 을 입력하여 NPM을 설치한다. (M1칩셋의 경우 15버전 이상부터 지원한다고 한다.)
  • npm install -g react-native-cli 를 입력하여 React-native-cli를 설치한다.

4. Xcode Install

  • App Store를 사용하여 설치한다. 아래의 사진과 같이 xcode를 검색하면 찾을 수 있다.

  • Xcode 설치가 완료되면 실행하여 커맨다 +, 키를 눌러 설정창의 Locations탭의 Command Line Tools를 설정해 준다.

5. Android Studio 설치

  • Android Studio 에 접속하여 다운로드 버튼을 클릭한다.

  • Mac OS의 칩셋에 따라 다운로드 받으면 된다.

  1. Preference -> Apperance & Behavior -> SDK Tools -> Android SDK Command-line Tools가 설치되어 있지 않다면 체크한 후 Apply버튼을 눌러 설치해 준다.
  2. SDK를 설치했다면 환경변수를 설정해야 한다.
  3. 터미널에 vi ~/.zshrc 또는 open ~/.zshrc를 입력하여 편집기를 열어준다.
  4. 아래의 환경변수를 입력해 준다. {}로 표기된 부분은 각 사용자에 맞게 적어준다.
  5. source ~/.zshrc 를 입력하여 환경변수를 적용해 준다.
# Android SDK
export ANDROID_HOME=/Users/{사용자이름}/Library/Android/sdk
export PATH=$PATH:$ANDROID_HOME/emulator
export PATH=$PATH:$ANDROID_HOME/tools export PATH=$PATH:$ANDROID_HOME/tools/bin
export PATH=$PATH:$ANDROID_HOME/platform-tools
  • M1 가상화 머신 대응가이드를 참조하여 가상화 머신을 생성하면 된다.

6. 프로젝트 생성

  1. 위의 준비 과정을 모두 마쳤다면, 프로젝트를 생성할 순서이다. 터미널을 열어 프로젝트를 생성할 경로로 이동하여 nvm use {노드버전} 을 입력한다.
  2. react-native init {프로젝트명}을 입력한다.
  3. /프로젝트경로/ios 로 이동하여 pod install 명령어를 입력해준다.
  4. Android 실행 명령어는 npm run android 또는 react-native run-android 이고, IOS 실행 명령어는 npm run ios 또는 react-native run-ios 이다.
